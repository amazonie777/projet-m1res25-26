# NovaSite — Projet GitHub Test

NovaSite est un petit site web moderne créé avec uniquement **HTML**, **CSS** et **JavaScript**.

Ce projet sert à tester un dépôt GitHub, s’entraîner à organiser des fichiers web, ou publier une première page avec **GitHub Pages**.

## Aperçu du projet

Le site contient :

- une page d’accueil moderne
- un menu responsive
- un bouton thème sombre / clair
- des animations au scroll
- des compteurs animés
- une section fonctionnalités
- une section projet
- un formulaire de contact visuel en JavaScript

## Technologies utilisées

- HTML5
- CSS3
- JavaScript Vanilla

Aucune librairie externe n’est nécessaire.
